# 班级日志系统 - 启动指南

## 端口配置

| 服务 | 端口 | 地址 |
|------|------|------|
| 前端 | 5183 | http://localhost:5183 |
| 后端 | 8010 | http://localhost:8010 |

---

## 后端启动

### 1. 进入后端目录

```bash
cd backend
```

### 2. 激活虚拟环境

```bash
.\venv\Scripts\activate
```

看到 `(venv)` 前缀表示激活成功。

### 3. 安装依赖（首次运行）

```bash
pip install -r requirements.txt
```

### 4. 数据库迁移（首次运行或模型变更后）

```bash
python manage.py migrate
```

### 5. 启动服务器

```bash
python manage.py runserver 8010
```

### 6. 退出虚拟环境（可选）

```bash
deactivate
```

---

## 前端启动

### 1. 进入前端目录

```bash
cd frontend
```

### 2. 安装依赖（首次运行）

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

启动成功后访问：**http://localhost:5183**

---

## 常见问题

### Q: 找不到 Django 模块？

确保已激活虚拟环境（终端显示 `(venv)` 前缀）。

### Q: 前端无法连接后端 API？

确保后端已启动在 8010 端口。

### Q: 数据库连接失败？

检查 PostgreSQL 服务是否启动。
